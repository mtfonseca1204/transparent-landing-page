# Transparent Landing Page

A premium, modern landing page for Transparent - a multi-layer liquidity intelligence platform.

## Features

- **Quadrant Navigation** - Smooth horizontal/vertical scrolling between 4 screens
- **Dark/Light Mode** - Toggle between themes with seamless transitions
- **Custom Cursor** - Interactive cursor with magnetic effects
- **Accordion Cards** - Interactive feature cards with descriptions
- **Blueprint Charts** - Professional SVG visualizations
- **Responsive Design** - Optimized for all screen sizes
- **Animated Loading Screen** - Clean progress indicator

## Structure

```
├── index.html          # Main landing page
├── public/
│   ├── Logo.svg        # Transparent logo
│   ├── blueprint-chart.svg  # Features section chart
│   └── chart23.svg     # Hero section chart
└── README.md
```

## Running Locally

Simply open `index.html` in a browser, or use a local server:

```bash
# Using Python
python3 -m http.server 8080

# Using Node.js
npx serve .
```

Then open `http://localhost:8080` in your browser.

## Deployment

This is a static HTML page that can be deployed to:

- **Vercel** - Drop the folder or connect the repository
- **Netlify** - Drag and drop deployment
- **GitHub Pages** - Enable in repository settings
- **Any static hosting** - Simply upload the files
